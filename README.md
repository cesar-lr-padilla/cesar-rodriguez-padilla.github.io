# Portfolio de César L. Rodríguez Padilla

Portfolio profesional para GitHub Pages inspirado en un diseño visual moderno tipo one-page.

## Publicación

1. Crear el repositorio `cesar-rodriguez-padilla.github.io`.
2. Subir todos los archivos de este paquete a la raíz del repositorio.
3. Ir a `Settings > Pages`.
4. Seleccionar `Deploy from a branch`.
5. Elegir `main` y `/root`.
6. Abrir `https://cesar-rodriguez-padilla.github.io`.

## Estructura

```text
index.html
404.html
README.md
assets/
  css/styles.css
  js/main.js
  img/favicon.svg
docs/
  CV_Cesar_Rodriguez_Padilla.pdf
```

## Personalización rápida

- Edita textos principales en `index.html`.
- Cambia colores en `assets/css/styles.css`, dentro de `:root`.
- Sustituye el CV en `docs/` manteniendo el mismo nombre de archivo.
