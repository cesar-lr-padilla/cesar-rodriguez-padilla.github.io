# Portafolio profesional - César L. Rodríguez Padilla

Portafolio estático listo para publicar en GitHub Pages.

## Archivos incluidos

- `index.html`: página principal.
- `assets/styles.css`: estilos responsive.
- `assets/main.js`: menú móvil y animaciones.
- `assets/favicon.svg`: icono del sitio.
- `docs/CV_Cesar_Rodriguez_Padilla.pdf`: CV descargable.

## Publicación en GitHub Pages

1. Crea un repositorio llamado:

```text
cesar_rodriguez_padilla.github.io
```

2. Sube todos los archivos de esta carpeta a la rama `main`.

3. En GitHub entra en:

```text
Settings > Pages
```

4. Selecciona:

```text
Deploy from a branch
Branch: main
Folder: /root
```

5. La web quedará publicada en:

```text
https://cesar_rodriguez_padilla.github.io
```

## Personalización rápida

Edita `index.html` para cambiar textos, enlaces a LinkedIn, GitHub o Miprochip.
